# Micro Detetive
Um jogo web educativo de microbiologia clínica desenvolvido em JavaScript puro, focado em raciocínio laboratorial, identificação de microrganismos e interpretação de testes microbiológicos.

## 📚 Projeto de extensão universitária

https://mayara-souza.github.io/Micro-Detetive/

---
## 🎮 Sobre o jogo

No **Micro Detetive**, o jogador assume o papel de um profissional de laboratório responsável por identificar microrganismos desconhecidos utilizando testes microbiológicos reais.

Cada rodada gera um caso diferente:

* um microrganismo é escolhido aleatoriamente
* suspeitos falsos são adicionados
* o jogador precisa interpretar os resultados laboratoriais corretamente
* escolhas ruins desperdiçam recursos laboratoriais

O objetivo é chegar ao diagnóstico correto utilizando o menor número possível de testes e evitando erros conceituais.

---
## 🧪 Funcionalidades

### ✅ Sistema procedural

Cada partida é diferente:

* microrganismos aleatórios
* suspeitos diferentes
* múltiplos caminhos diagnósticos

---

### ✅ Sistema de testes microbiológicos

#### Bactérias

* Coloração de Gram
* Catalase
* Oxidase
* Coagulase
* Morfologia bacteriana

#### Fungos

* Micromorfologia
* Tubo germinativo
* Urease
* Azul de lactofenol

---

### ✅ Sistema de raciocínio laboratorial

O jogo avalia:

* interpretação dos resultados
* eliminação correta de suspeitos
* estratégia diagnóstica
* escolha apropriada dos testes

---

### ✅ Sistema de penalidade

O jogador pode perder moedas ao:

* escolher testes inadequados
* eliminar microrganismos incorretamente
* desperdiçar recursos laboratoriais
* aplicar testes bacterianos em fungos (e vice-versa)
